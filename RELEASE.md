# v1.0.0 发布

主要功能：

*   **题目提取**: 支持从雨课堂测验和试卷页面一键提取所有题目。
*   **PPT 提取**: 支持提取雨课堂课件中的 PPT 幻灯片，并提供拼接长图功能。
*   **AI 解答**: 集成 AI 解答功能，支持单个题目或批量处理。
*   **自定义 AI 配置**: 允许用户自定义 API URL、Key 和模型。
*   **富文本渲染**: 支持 Markdown、数学公式和代码高亮。
*   **多布局切换**: 提供多种视图布局以适应不同场景。
*   **辅助工具**: 提供“复制全部文本”和“导出所有图片”功能。

项目地址：https://github.com/soundstarrain/yuketang-assistant
